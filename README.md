## Áreas que desarrollo

* Desarrollo de plataformas web con **React, Next.js y Django**.
* Sistemas con autenticación, roles, dashboards y paneles administrativos.
* APIs REST con Django y consumo desde frontend.
* Despliegue de proyectos con **Docker, Nginx, Gunicorn y PostgreSQL**.
* Sitios institucionales, bibliotecas digitales y páginas informativas.
* Automatización de reportes, formularios y flujos de trabajo.
* Integración con herramientas como **Typeform, KoboToolbox, Google Apps Script y GA4**.
* Mapas interactivos, visores territoriales, **GeoServer, PostGIS y datos geoespaciales**.
* Diseño de interfaces visuales, limpias y funcionales.
